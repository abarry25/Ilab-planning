How to use this: each row is a program or task pulled from the AY26–27 planning tool — click any name to rename it, use + Add row at the bottom of a section to add a new one, and hover a row to reveal the × to remove it (removals show an Undo for a few seconds). In day view, click individual days; use Collapse to weeks to mark a whole week at once (click again to expand back to days — your day-level marks are preserved either way). Type initials into the small box for who owns it, and use the notes field for cadence or context. Check the box on any row and click Create ClickUp export to generate one line per marked date, ready to edit and download as a CSV for import. Milestones pinned above the grid come from the FAS Registrar's published calendar — double-check exact dates before locking programming around them. Switch the Fall 2026 / Spring 2027 tabs at the top to plan the next term whenever you're ready; each term keeps its own board. Group tags: MACRO = ecosystem events to track, FUNC = an operating rule rather than an event, CORE = flagship i-lab programming, SERVICE = supporting programs and partnerships.
